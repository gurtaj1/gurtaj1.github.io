KING OF VAPE

Deployed Website

The Front-End of an eCommerce Site specialising in Vape Goods
Technologies: ReactJS, React-Router, React-Redux, Bootstrap, CSS
Background: Made for King of Vape's Company Website

This is my favourite creation to date. The task was to Develop the Front-End for an eCommerce website.

I found that ReactJS worked really well for cutting out large amounts of repetetive work. All of the product lists, and indivdual product pages were essentially just the same components but with different props. I utilised React-Redux to store all the information on the products and to allow for the ability to filter product types as well as to search through products using keywords.

Due to the lack of a Back-End Developer, the company eventually opted to use a well known eCommerce platform (also setup by myself). I hope to come back to this project one day, when I have more knowledge on Back-End technologies, and offer it as a fully bespoke option.

MEMORY GAME

GitHub Repository
Deployed Page

Recreation of the Simon Game, popular in the 70's and 80's
Technologies: HTML, CSS, JavaScript, jQuery
Background: Created as one of my projects for Free Code Camp's Front End Development Program

This project took a lot of extra thought and careful planning with regards to function and ease of use.

The toughest part of this project was probably the need for audio feedback. I wanted the audio to sound for as long as any of the buttons where 'held down', since this is how the Simon Game bahaves in real life. I researched several different ways to do this with the mp3 files that were given in the project outline, but in the end I decided that creating my own frequencies would make for a better user experience. After a little research and trial-and-error, I had successfully managed to create a web audio API context within the JavaScript code and assigned a frequency to each colour button within the game.

If I was to go back to this project now, I would look at optimising it across more browsers since, currently, sounds are not being reproduced as expected on the mobile version of chrome.

POMODORO CLOCK

GitHub Repository
Deployed Page

Emulation of a Pomodoro Clock
Technologies: HTML, CSS, JavaScript, jQuery
Background: Created as one of my projects for Free Code Camp's Front End Development Program

This project required a lot of consideration into the actual use of a real-life pomodoro clock.

The toughest part of this project was actually in deciding what kind of functionality (besides starting a countdown, and then waiting for it to alarm) and end-user would need in such a device. I made three different versions of the clock but in the end I stuck with the one that can interchange between session and break periods, and change timer settings during a countdown, without affecting the current countdown.

If I was to go back to this project now, I would alter the code to use functions that rely on date methods rather than setTimeout(), due to the inaccuracies associated with it.

NOUGHTS AND CROSSES

GitHub Repository
Deployed Page

Noughts and Crosses game with Unbeatable AI
Technologies: HTML, CSS, JavaScript, jQuery
Background: Created as one of my projects for Free Code Camp's Front End Development Program

This project showed me the level of sophistaction that is associated with even the simplest of algorithms.

This project was initially completed without a great deal of research being performed. Even though I was very impressed with the fact that I had managed to do this, I was not happy about the fact that my AI was so easy to beat (chose empty slots at random). It was for this reason that I decided to work at my project some more before submitting it. I learnt about the Minimax Algorithm and in doing do I got a glimpse of the true beauty in programming. I then had nothing left to do but to implement the algorithm into my own code.

If I was to go back and do this project again, the only thing I change would be the styling, particularly the fonts used, and the re-sizing of divs when viewing on different screen sizes.

CALCULATOR

GitHub Repository
Deployed Page

Non-Scientific Calculator
Technologies: HTML, CSS, JavaScript, jQuery
Background: Created as one of my projects for Free Code Camp's Front End Development Program

This project allowed me to put my coding skills into practice, whilst excercising my knowledge of basic calculator functionality.

This project was coded executed using only my core knowledge and some trial-and-error. I did read-up on some jQuery and CSS notation, but only to jog my memory, the logic applied was all my own. I state this, not for the sake of being boastful, but rather to show how this project was very important in solidifying my confidence as a developer, and in motivating me to push further.

One thing I would change if I was to do this project again, would be the precision of the outputted answer. Currently, the answer's precision is based on that if the most precise number entered into the sum. Instead I would base the precision on the amount of screen space available, and the difference between the 'exact' answer, and its closest integer value.
